# Huoshen Codex Pet

Huoshen（火神）is an irritable but reliable pixel-art fire demon companion for Codex.

## Character

- Style: Pixel art
- Personality: Irritable, energetic and focused
- Main colors: Orange, yellow and charcoal black
- Features:
  - Large yellow-and-black striped horns
  - Small forehead horns
  - Black bat wings
  - Spiked collar
  - Flame-shaped claws
  - Smoky lower body and pointed tail

## Animation states

- idle
- running-right
- running-left
- waving
- jumping
- failed
- waiting
- running
- review

## Installation

Clone or download this repository, then copy the package into:

```text
~/.codex/pets/huoshen/
```
